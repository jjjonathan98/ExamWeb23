F1 API Documentation

Overview

The F1 API allows for managing drivers and teams in a Formula 1 context. This documentation provides an overview of the available endpoints and their usage.

Base URL

The base URL for all API endpoints is http://localhost:5089/api/. All endpoint paths should be appended to this base URL.

Drivers

GET /api/Drivers - Retrieves a list of all drivers.

POST /api/Drivers - Creates a new driver.

PUT /api/Drivers - Updates an existing driver.

GET /api/Drivers/{id} - Retrieves a driver by ID.

DELETE /api/Drivers/{id} - Deletes a driver by ID.

GET /api/Drivers/GetByNationality/{nationality} - Retrieves drivers by nationality.

ImageUpload

POST /api/ImageUpload - Uploads an image file.

To upload an image file, send a POST request to the /api/ImageUpload endpoint. Include the image file as part of the request payload. The response will contain information about the uploaded image.

Teams

GET /api/Teams - Retrieves a list of all teams.

POST /api/Teams - Creates a new team.

PUT /api/Teams - Updates an existing team.

GET /api/Teams/{id} - Retrieves a team by ID.

DELETE /api/Teams/{id} - Deletes a team by ID.

GET /api/Teams/GetByTeam/{team} - Retrieves teams by team name.

Schemas

Drivers

Schema definition for the Drivers object:

{ "id": number, "name": string, "nationality": string, ... }

Teams

Schema definition for the Teams object:

{ "id": number, "name": string, "country": string, ... }
